### 活用 Swift 类型推断
* 类型推断的工作机制
```
let x = ""
//虽然代码中没有显式的写出 x 的类型，但 compile 可以通过类型推断算法将其补全为下面的代码。
let x: String = ""
```
* compiler 如何自动捕获代码中的错误
    - 记录源码中的错误信息。
    - 尝试去修复错误一遍继续类型推断算法。
    - 基于收集的信息来提供有用的错误信息。
* 类型推断就像拼图，一块一块的拼完整。
* 显示当前错误信息：
    - Xcode -> Behaviors -> Edit Behaviors -> Fails -> 选中 [Show] Navigator [Current]
