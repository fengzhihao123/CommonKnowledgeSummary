## 27.移除元素

### 题目介绍
```
给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。
不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地 修改输入数组。
元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

示例 1:
给定 nums = [3,2,2,3], val = 3,
函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。
你不需要考虑数组中超出新长度后面的元素。
示例 2:
给定 nums = [0,1,2,2,3,0,4,2], val = 2,
函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。
注意这五个元素可为任意顺序。
你不需要考虑数组中超出新长度后面的元素。
```

### 思路
```
 题目关键字：原地移除、无需考虑数组超出新长度后面的元素
 思路：
 1、遍历数组，记录当前值不等于val的位置 - i
 2、若值不等于val，则将 j 位置的元素赋值给 i
 3、遍历结束，返回 i
```
### 代码
```
func removeElement(_ nums: inout [Int], _ val: Int) -> Int {
    var i = 0
    for j in 0..<nums.count {
        if nums[j] != val {
            nums[i] = nums[j]
            i = i + 1
        }
    }
    return i
}
```

```

第二次遍历：i = 1，j 位置的元素为 1，不等于 2，进入if， num[1] = num[1] -> nums = [0,1,2,2,3,0,4,2]， i = 2
第三次遍历：i = 2，j 位置的元素为 2，等于2，不进入if -> nums = [0,1,2,2,3,0,4,2]， i = 2
第四次遍历：i = 2，j 位置的元素为 2，等于2，不进入if -> nums = [0,1,2,2,3,0,4,2]， i = 2
```
